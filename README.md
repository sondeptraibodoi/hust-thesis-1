# Đồ án 1
# set up
tải php, composer, node

---
backend
- tạo database
- chạy lệnh
 + cd vào thư mục server
 + composer install
 + composer dump
 + kết nối database trong file env
 + php artisan optimize
 + php artisan migrate
 + php artisan db:seed
 + php artisan serve

---
client
- chạy lệnh
 + cd vào thư mục client
 + npm i
 + npm run dev
